Get started with the recruitment APIs

What does an API do?

APIs (Application Programming Interfaces) let applications share information and data with each other.

If you connect using the Recruitment API, you can use your own systems to publish to Find an apprenticeship. 

You can also use the Display adverts API to retrieve vacancies from Find an Apprenticeship and display on your websites.

How do I use the API?

You'll need someone with technical knowledge to connect your system with the API, for example a software developer.

You can share the developer get started page with the person who is going to connect your system with the API, they can access the API from there. 

It can take a while to connect using an API, so leave enough time to integrate it.

How do I access the API?

If you have owner permissions you can access the API keys. 

Your developer will be able to try out the API using the sandbox provided.

You will need to provide your developer with a key and the developer get started page so they can use the APIs.
