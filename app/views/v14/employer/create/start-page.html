Create an advert

Recruit an apprentice by creating an advert to post on Find an apprenticeship (opens in a new tab).

Before you start

Before you create an advert, you’ll need to know:

  • the name of the apprenticeship training course and training provider - if you haven't found training provider yet, use Find apprenticeship training for your apprentice (opens in a new tab)
  • the duration of the apprenticeship
  • the number of hours spent working and training
  • how much your apprentice will be paid
If you don’t pay the apprenticeship levy, you’ll need to reserve apprenticeship funding before creating an advert.
