Create an advert

Recruit an apprentice by creating an advert to post on Find an apprenticeship (opens in a new tab).

Before you start

Before you create an advert, you’ll need to know:

  • the name of the apprenticeship training course
  • the name of the training provider - if you don’t have a training provider yet, you can find a provider (opens in a new tab)
  • the duration of the apprenticeship
  • the number of hours spent working and training
  • how much your apprentice will be paid
You can use our recruitment Application Programming Interfaces (APIs) (opens in a new tab) to create adverts using your existing systems or display adverts from Find an apprenticeship on your websites.
